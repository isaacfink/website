Django is a web applciation framework for the [Python](/stack/python) programming language, despite being one of the oldest web frameworks it is still very popular and used for many applicaitons like instagram, mozilla and spotify

Django is a batetries included framework which means it comes standard with a lot of features needed for web applications, some of the built in features are

1. Object Relational Mapper (ORM)
2. Templating Engine
3. Form Validation
4. Authentication
5. Web based admin panel

And is one of the best choices to quickly build applciations, django has a very active community of maintainers and contributors and is very well documented, using python it also means that it is easy to use and the code is easy to understand
