JavaScript or JS is consistently ranked as one of the most popular programming languages, originally designed to add interactivity to web applications it is nowadays used all over for a veriety of different use cases, we can build web applications with express, mobile applications with react native, desktop applications with electron and even server side applications with nodejs.

JavaScript is a weak typed dynamic programming language with support for a veriety of different paradigms, it is a prototype based object oriented language with support for functional programming as well as object oriented programming.
