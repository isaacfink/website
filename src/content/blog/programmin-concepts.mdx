## Data Structures and Algorithms

### Big o notation

Big o notation is a way to define the speed and/or memory usage of an algorithm. It is useful when comparing algorithm and deciding which ones are better for a specific task, there are multiple ways to define a big o notation, you could choose the best case scenario, worst case scenario, average, median and many different methods but in general we aim for a good big o score for the worst case scenarios
